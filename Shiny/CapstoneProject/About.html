Welcome to Next Word Prediction Application

This application is a next word prediction. The sample data is downloaded from the coursera site. The data consists of the twitter, blogs and news data.

The n-grams were generated using the 'tm' and the 'quanteda' packages. The sample data is processed to remove urls, special characters, punctuation, numbers, stopwords and curse words. The bi-grams and tri-grams are generated with skip grams of 0:3 for 10% of the sample data. The n-grams data is further processed by splitting the data into current word and next word with their counts. This format is basically markov chains in the simplest form. This final data is stored in the bigrams-markov.rds and trigrams-markov.rds so that we are only using the results for this shiny application.

The user types the word in the input box and the next 10 words based on frequencies are predicted in the drop down list. The user can select the next word and the input is updated with the word and the search is run again and the next 10 words are populated in the drop down list. This process is repeated over and over.

The word cloud plot is generated from the cached data. The 'Minimum Frequency' a word appears and the 'Maximum Number of Words' variables are used to re-plot the data visualization.

The 'Calculate Next Words' button runs the search based on what has been typed so far to predict the next 10 words.")
